# electric-circuit
Interactive Ohm’s Law Circuit Demo is a single-file HTML/CSS/JavaScript educational simulation built with inline SVG and no external libraries. It is designed for high school physics teaching and demonstrates the basic relationship between voltage, resistance, current, and power in a simple DC circuit.

The demo includes a battery, resistor, wires, switch, and animated current flow. Users can adjust the voltage and resistance with sliders and immediately see the effect on current and power using Ohm’s law: I = V / R and P = VI. A visual resistor glow increases with power, helping students connect the numerical values to an intuitive physical effect.

Because the project is fully self-contained in a single HTML file, it is easy to deploy, share, and run in any modern browser without installation, frameworks, or build tools. This makes it useful for classroom demonstrations, LMS uploads, student self-study, and simple static hosting through GitHub Pages, Netlify, or similar platforms.

Features:

Single-file standalone demo

Pure HTML, CSS, JavaScript, and SVG

No external libraries or dependencies

Adjustable voltage and resistance

Live current and power calculations

Open/close switch interaction

Animated current visualization

Classroom-friendly design for introductory physics

This project is intended as a clear visual teaching aid rather than a fully detailed electrical engineering simulator.
